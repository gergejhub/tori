# Lilike – Történelem gyakorló (Kereszténység + középkori egyház)

Ez a repo egy **statikus, GitHub Pages-en futtatható** gyakorló app (vanilla JS), amivel témazáróra lehet készülni.

## Tananyag
- **Tankönyvi oldalak:** 58–81 (a csatolt „Történelem – Kereszténység” beszkennelt PDF-ek alapján)
- Témák: Bibliai alapok és Ószövetség, Jézus élete és tanításai, a kereszténység terjedése/üldözései, Konstantin fordulata, középkor alapjai, várak, középkori egyház, szentek/ereklyék, szerzetesrendek, román és gótikus stílus, oktatás/kódexek/egyetemek.

## Használat
- Nyisd meg az `index.html`-t böngészőben (vagy deploy GitHub Pages-re).
- Válassz módot: **Gyorskártyák / Kvíz (választós) / Kvíz (beírás) / SRS**
- A *Kevert* mód mindent kever, a témaszűrő pedig tagek alapján szűr.

## Adatok

- A statisztika és SRS helyben, a böngésző `localStorage`-ában tárolódik (kulcs: `lilike_tortenelem_keresztenyseg_v1`).
